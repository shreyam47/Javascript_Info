====DEBUGGING IN BROWSER====

1) Sources Panel
    - turn on developer tools in chrome
    - select "sources" panel
    - select JS file
    - edit JS file

2) Breakpoint
    - debugger will automatically pause
    - can type "debugger;" in JS file

3) Console.log
